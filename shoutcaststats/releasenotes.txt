<h1>Initial Release. Version 05.01.00</h1>
<p> This module renders the statistic informations from a Shoutcast broadcasting server.  
Additinal it could added Links to different messengers with direct opening a messenger window to chat with the current DJ.
Also you could provide Links to open the Stream on different players of the client. This module inscludes 2 module definations, 
whish you could set up with different permissions.
 </p>
<p>
Minimum DotNetNuke version to use 05.05.00
</p>
<p>All Features could be easy setup over the Settings.</p>
<h2>Features</h2>
<ol>
<li>Station Name</li>
<li>Current Listeners</li>
<li>Listener Peak</li>
<li>Max. Listeners</li>
<li>Current Song</li>
<li>DJ Name</li>
<li>AIM, AOL, ICQ or MSN and Yahoo Messenger Links to DJ</li>
<li>Bitrate</li>
<li>Codec Type</li>
<li>4 Player Links</li>
</ol>
<ol>
<li>View count of the XML file</li>
<li>Listerner statistics list</li>
<li>Last played songs list</li>
</ol>
